音频纪要生成工具

Whisper 本地转录 · AI 生成纪要 · 保存到指定目录

0 打开 Terminal（终端）

按 Command ⌘ + 空格，输入 Terminal，回车即可打开。
或者：Finder → 应用程序 → 实用工具 → 终端
⚠ 本工具依赖 Homebrew（brew），仅支持 macOS。Windows 用户需先安装 WSL。

1 安装 Python 3

先检查：python3 --version
如未安装：
brew install python3

2 安装依赖（只需一次）

brew install ffmpeg
pip3 install openai-whisper openai

3 获取 API Key（任选一个）

Groq（免费）：console.groq.com
OpenAI：platform.openai.com
DeepSeek：platform.deepseek.com
通义千问：dashscope.console.aliyun.com
Claude（Anthropic）：console.anthropic.com
使用 Claude 需额外安装：pip3 install anthropic

4 启动服务

必须先 cd 进入文件所在的文件夹，再运行： 
cd 你的文件夹路径
python3 server.py
如何获取文件夹路径？
在 Finder 中找到 MeetingNotes-Share 文件夹，
按住 Option ⌥ 再右键点击文件夹 → 选择 "将…拷贝为路径名称"
然后把复制的路径粘贴到 cd 后面。
也可以直接把文件夹拖入 Terminal 窗口，路径会自动填入。
浏览器自动打开。使用期间保持终端窗口开着，关掉服务就停了。

0 Open Terminal

Press Command ⌘ + Space, type Terminal, press Enter.
Or: Finder → Applications → Utilities → Terminal
⚠ This tool uses Homebrew and is macOS-only. Windows users need WSL.

1 Install Python 3

Check first: python3 --version
If missing: 
brew install python3

2 Install dependencies (once)

brew install ffmpeg
pip3 install openai-whisper openai

3 Get an API key (pick one)

Groq (free): console.groq.com
OpenAI: platform.openai.com
DeepSeek: platform.deepseek.com
Qwen: dashscope.console.aliyun.com
Claude (Anthropic): console.anthropic.com
To use Claude, also run: pip3 install anthropic

4 Start the server

cd into the folder first, then run: 
cd your/folder/path
python3 server.py
How to get the folder path:
In Finder, locate the MeetingNotes-Share folder,
hold Option ⌥ and right-click it → select "Copy … as Pathname"
then paste it after cd.
Or drag the folder into the Terminal window — the path fills in automatically.
Browser opens automatically. Keep the Terminal window open — closing it stops the server.
